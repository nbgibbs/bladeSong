Welcome to the Blade Song rule book! 

This is a work in progress.

Blade Song is a roleplaying game system developed from Cogent Roleplay (cogentroleplay.com). This rule book will (eventually) contain everything needed to play, but I wanted to give credit to the original idea. 

The majority of rolls in the system consist of rolling a dice pool of a number of dice each with a %50 chance of being a win (a d6 counting 4, 5 and 6 as a win). The dice pool for a given challenge is:

3 + any applicable [[attributes]] scores (could be negative) + any applicable skills + any tools or other modifiers.

More skills and tools = more dice = better chance of getting enough win against a given challenge.
